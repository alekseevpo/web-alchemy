Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [2025-01-XX] - Последние изменения

### Изменено
- Заголовок секции услуг "Что мы делаем" → "Наши проекты" (для всех языков: ru, en, es)
- Заголовок "Наши проекты" выровнен по центру
- Возвращена кнопка "назад" в слайдер (навигация в обе стороны)

### Исправлено
- Настройка слайдера на бесконечную прокрутку в обе стороны
- Начальный слайд установлен на середину (индекс 2 из 5)

## [2025-01-XX] - Слайдер и навигация

### Добавлено
- 3D эффект для слайдера услуг (перспектива, поворот, масштаб, размытие)
- Два новых блока услуг: "Корпоративные сайты" и "Техническая поддержка"
- Слайдер с индикаторами и кнопками навигации для всех устройств
- Фиксированная высота слайдов на десктопе (524px)

### Изменено
- Слайдер теперь работает на всех устройствах (ранее был только на мобильных)
- Убрано увеличение слайдов при наведении
- Навигационные кнопки размещены ближе к краям слайдера

## [2025-01-XX] - UI/UX улучшения

### Изменено
- Размытие фона тёмной темы увеличено
- Затемнение фона тёмной темы уменьшено в два раза
- Кнопка меню мобильной версии сделана квадратной, размер как у кнопки темы
- Кнопка меню перемещена справа от кнопки выбора языка

## [2025-01-XX] - SEO оптимизация

### Добавлено
- Open Graph метатеги для социальных сетей
- Twitter Card метатеги
- JSON-LD структурированные данные (Schema.org):
  - Organization (организация)
  - WebSite (веб-сайт)
  - Service (услуги)
- Файл `robots.txt` с директивами для поисковых систем
- Динамический `sitemap.xml` со всеми страницами сайта
- Канонические URL для всех страниц
- Hreflang метатеги для многоязычности
- Динамические метаданные для всех страниц

### Улучшено
- Метаданные страниц оптимизированы для поисковых систем
- Добавлены альтернативные языковые версии страниц

## [2025-01-XX] - Интеграции и аналитика

### Добавлено
- Vercel Analytics для отслеживания трафика
- Vercel Speed Insights для мониторинга производительности

## [2025-01-XX] - Мобильная версия

### Исправлено
- Улучшена мобильная навигация
- Оптимизировано размещение кнопок меню и навигации
- Улучшены touch targets для мобильных устройств

## [2025-01-XX] - Контент и локализация

### Изменено
- Обновлён hero subtitle с более подробным описанием услуг
- Обновлены переводы для всех языков (ru, en, es)

## [2025-01-XX] - Cookie Consent и правовые страницы

### Добавлено
- Система согласия на использование cookies с баннером
- Страница "Политика конфиденциальности" (`/privacy`)
- Страница "Условия использования" (`/terms`)
- Страница "Политика cookies" (`/cookies`)

### Изменено
- Футер дополнен ссылками на правовые документы
- Добавлена информация о компании и социальных сетях в футере

## [2025-01-XX] - Преобразование проекта

### Изменено
- Преобразование проекта из технического задания в сайт компании Web Alchemy
- Замена всего контента на информацию о компании
- Обновление навигации и структуры сайта

### Добавлено
- Новые страницы:
  - `/projects` - Проекты
  - `/services` - Услуги и цены
  - `/feedback` - Обратная связь и отзывы
  - `/faq` - Часто задаваемые вопросы
  - `/support` - Поддержка
  - `/developers` - Наша команда (переработана)

### Удалено
- Весь контент технического задания
- Левое навигационное меню в десктопной версии

## Легенда

- **Добавлено** - новые функции
- **Изменено** - изменения в существующей функциональности
- **Устарело** - функции, которые скоро будут удалены
- **Удалено** - удалённые функции
- **Исправлено** - исправления багов
- **Безопасность** - исправления уязвимостей
