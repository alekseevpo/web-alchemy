# Руководство по созданию OG изображения

## Требования

- **Размер:** 1200x630 пикселей (соотношение 1.91:1)
- **Формат:** PNG или WebP (рекомендуется WebP для меньшего размера)
- **Размер файла:** < 200KB (желательно < 100KB)
- **Имя файла:** `og-image.jpg` или `og-image.webp`
- **Расположение:** `/public/og-image.jpg` или `/public/og-image.webp`

## Содержание изображения

OG изображение должно содержать:
- **Логотип Web-Alchemy** (центрирован или слева)
- **Название компании:** "Web-Alchemy"
- **Подзаголовок:** "Разработка Веб-приложений и сайтов любой сложности"
- **Цветовая схема:** Соответствует бренду (можно использовать цвета из сайта)

## Варианты создания

### Вариант 1: Онлайн инструменты

1. **Canva** (https://www.canva.com/)
   - Создайте дизайн 1200x630px
   - Используйте шаблон "Open Graph Image"
   - Экспортируйте как PNG или WebP

2. **Figma** (https://www.figma.com/)
   - Создайте фрейм 1200x630px
   - Используйте логотип из `/public/logo.png`
   - Экспортируйте как PNG

3. **Adobe Express** (https://www.adobe.com/express/)
   - Шаблон "Social Media Post" 1200x630px
   - Добавьте логотип и текст

### Вариант 2: Использование HTML шаблона

Создан файл `og-image-template.html` в корне проекта. Вы можете:
1. Открыть его в браузере
2. Сделать скриншот (1200x630px)
3. Или использовать инструменты типа Puppeteer для автоматической генерации

### Вариант 3: Использование существующего логотипа

Можно использовать существующий логотип `/public/logo.png` и добавить текст через графический редактор.

## Рекомендации по дизайну

- **Фон:** Светлый или темный градиент, соответствующий теме сайта
- **Текст:** Читаемый, контрастный
- **Логотип:** Четкий, хорошо видимый
- **Минимализм:** Не перегружайте изображение деталями

## После создания

1. Сохраните файл как `og-image.jpg` или `og-image.webp` в папку `/public/`
2. Обновите `app/layout.tsx` если используете другой формат:
   ```typescript
   images: [{
     url: '/og-image.webp', // или .jpg
     width: 1200,
     height: 630,
     alt: 'Web-Alchemy - Разработка веб-приложений',
   }]
   ```

## Оптимизация

После создания изображения, оптимизируйте его:

```bash
# Используя ImageOptim (macOS)
# Или онлайн: https://squoosh.app/
# Или через sharp в Node.js
```

После добавления изображения, проверьте его через:
- **Facebook Debugger:** https://developers.facebook.com/tools/debug/
- **Twitter Card Validator:** https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector:** https://www.linkedin.com/post-inspector/
